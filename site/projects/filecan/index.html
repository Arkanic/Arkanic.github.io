Filecan

A node.js file uploader

Features:

    • Multiple file uploads
      Optional password required to upload files
      Customizable filenames
      No html page for the files - redirects directly to the raw file
      Good for sending files to discord, allows a way to go past the 8mb file limit
      wget and cURL compatible

    • Example
