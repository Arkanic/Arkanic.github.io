Exdroid

(A node.js top-down shooter)

About

Exdroid is a little shooter game i've been semi-working on for the past half a year, roughly based off Victor Zhou's example .io game.

It uses socket.io for server-client communication, and plain old canvas for the rendering.

GitHub: https://github.com/Arkanic/Exdroid
